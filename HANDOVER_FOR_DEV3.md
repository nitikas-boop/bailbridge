# 🚀 BailBridge Frontend Integration Guide (Developer 3)

Welcome to the BailBridge handoff package. This document contains everything you need to connect the React/Next.js frontend to the Python/FastAPI backend.

## 🌐 API Endpoints

### 1. **Case Management**
- **POST `/api/start-case`**
  - **Purpose**: Initialize a new bail assistance journey.
  - **Payload**: `{ "user_id": "string", "defendant_name": "string" }`
  - **Response**: `{ "case_id": "GUID", "status": "intake_started" }`

- **GET `/api/case-status/{case_id}`**
  - **Response**: Current status, metadata, and PDF URLs.

### 2. **Sahayak Script**
- **GET `/api/sahayak-script/{case_id}`**
  - **Purpose**: Fetch the 3-point verbal script in English + Regional Language.
  - **Response**: `{ "sahayak_script": "..." }`

## 🔌 WebSocket Events (Socket.IO)
**URL**: `ws://<backend-url>/ws/socket.io`

### **Listen for (Incoming from Backend)**
| Event | Payload | Purpose |
| :--- | :--- | :--- |
| `connected` | `{ "sid": "..." }` | Connection confirmation. |
| `new_message` | `{ "text": "...", "sender_type": "ai/agent" }` | Real-time chat messages (includes translations). |
| `agent_typing` | `{ "is_typing": bool }` | Show/hide the typing bubble. |
| `case_update` | `{ "status": "...", "bail_pdf_url": "..." }` | Triggers PDF download or dashboard refresh. |

### **Emit to (Outgoing to Backend)**
| Event | Payload | Purpose |
| :--- | :--- | :--- |
| `join_case` | `{ "case_id": "..." }` | Join the real-time room for a specific case. |
| `defendant_message` | `{ "case_id": "...", "text": "..." }` | User answers to intake questions or chat. |
| `file_uploaded` | `{ "case_id": "...", "file_name": "..." }` | Trigger a PDF refresh after evidence upload. |

## 📂 Core Firestore State Keys
Use these keys to sync your frontend state with the `cases` collection:

- `preferred_language`: Set automatically by Jatayu. Use this to toggle UI language.
- `intake_step`: Current question index (0 to 7).
- `bail_strategy`: The determined legal classification (e.g., "Mandatory Bail").
- `bail_pdf_url`: Direct link to the generated Draft Bail Application.
- `sahayak_script`: The generated court verbal script.
